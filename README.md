- Algotirmos;
- Estruturas Condicionais e de Repetição;

- Introdução à HTML5 e CSS3;
- D.O.M.;
- Banco de Dados Relacional;
- MySQL;
- P.O.O.;
- Spring Boot Framework;
- CRUD em uma API;

Foram desenvolvidos alguns projetos em Spring Web e por meio da API.
